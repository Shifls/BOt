# Опис проєкту

Метою даного проєкту є створення консольного чат-бота мовою **Python**, який допомагає користувачу організовувати події.

Бот дозволяє:

- додавати події  
- переглядати список подій  
- фільтрувати події  
- редагувати події  
- видаляти події  
- шукати події  
- знаходити найближчу подію  

Дані зберігаються у **JSON-файлі**, тому після перезапуску програми події не зникають.

---

# Структура даних

Події зберігаються у файлі **events.json**.

Приклад події:

```json
{
  "title": "Екзамен з програмування",
  "date": "2026-05-20",
  "start_time": "10:00",
  "end_time": "12:00",
  "category": "навчання"
}
```

Кожна подія містить:

- **title** — назва події  
- **date** — дата події  
- **start_time** — час початку  
- **end_time** — час завершення  
- **category** — категорія або опис  

---

# Алгоритм роботи бота

1. Після запуску бот зчитує події з файлу **events.json**.  
2. Користувачу показується привітання.  
3. Бот очікує введення команди.  
4. Залежно від команди виконується одна з функцій:
   - додавання події
   - перегляд подій
   - пошук
   - редагування
   - видалення
5. Після змін інформація зберігається у файл.
